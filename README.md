# Graphs: Beginner to Pro Matplotlib

![Graphs](https://www.eurokidsindia.com/blog/creative-ways-to-teach-graphs-and-charts-to-preschoolers.php)

This repository contains step-by-step Jupyter Notebooks designed to take you from a Matplotlib beginner to an advanced user.

## 🚀 Getting Started

### Prerequisites
Make sure you have Python installed, then install the required libraries:
```bash
pip install matplotlib notebook
```

### Repository Structure
* **[Plot(LEVEL1).ipynb](./Plot(LEVEL1).ipynb)**: Introduction to basic plots (Line, Bar, Scatter).
* **[Plot(LEVEL2).ipynb](./Plot(LEVEL2).ipynb)**: Advanced formatting, markers, and multi-plot layouts.

## 🛠️ How to Use
1. Clone this repository.
2. Open your terminal and run `jupyter notebook`.
3. Open the level you wish to study and run the cells.
